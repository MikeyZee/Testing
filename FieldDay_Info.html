Betaflight Configuration

BlackBox logging settings

Betaflight's blackbox logging system works effciently with FieldDay's algorthms. Many modern flight controllers have integrated barometers, GPS chips, gyroscopes and other sensors which can be utilised by FieldDay. Betaflight has produced a software which allows some interpretation of blackbox data and this software is required for FieldDay to function properly. This software - (Blackbox decoding tool) can be found below.

Betaflight Tools Download Mac & Windows



A tutorial demonstrating how to use the decoding tool to generate csv files for use within FieldDay can be found on the right handside of the page.
With further iterations, an integrated batch script will fill in all values after the CSV file is uploaded to FieldDay's servers. For the moment, manual input is required.
